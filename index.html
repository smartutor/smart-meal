📧 Google Apps Script 自動寄信
系統每天早上 06:00 自動檢查餘額，低於 $200 且尚未通知的學生將自動收到 Email。
儲值後會自動重置通知狀態，下次餘額再度不足時才會再寄。

設定步驟：
1. 前往 script.google.com 新增專案
2. 將下方程式碼貼入並儲存
3. 設定觸發條件：每天 05:00–06:00 執行
4. 首次執行時授權 Gmail 權限

說明
每位學生有專屬代碼，進入學生詳情可複製代碼或連結給家長。
家長輸入代碼後只能查看自己孩子的資料，無法編輯任何內容。

想修改系統名稱、配色、封面圖片或 Logo，請用文字編輯器打開 HTML 檔案，修改最頂端的 CONFIG 區域。
